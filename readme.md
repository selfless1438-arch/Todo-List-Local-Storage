# 📝 Todo List App

A simple and responsive Todo List web application built using **HTML, CSS, and JavaScript**.  
This app allows users to manage daily tasks with data stored in **LocalStorage**.

---

## 🚀 Features

- ✅ Add new tasks  
- 🌗 Dark & Light with shortcut key.  
- 🗑️ Delete tasks  
- ✔️ Mark tasks as completed  
- 💾 Persistent storage using LocalStorage   
- 🔢 Task counter  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- LocalStorage API
